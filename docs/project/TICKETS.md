### SPEU-002 — Падлік праслухоўванняў

**Стан:** `review`. У БД: `listen_outcomes` (user / `anon_listener_id`), агрэгацыя **`track_listen_by_day_minsk`** з палямі `full_total` і `partial_total`; дашборды label/artist/**per-track** праз RPC. Застаецца: пацвердзіць адпаведнасць прадуктавым парогам (≥30 с, поўнае праслухоўванне), паводзіны пры seek, апісаць для каманды ў адным месцы (каментар да RPC або `ListenMethodologyDialog`).

**AC:** парогі ў кодзе/RPC; без дубляў; мінімум ПД для анонімаў.
