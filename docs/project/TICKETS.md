**Стан:** `done` (2026-04-06). Табліца `speu.track_likes`, toggle праз API, мадалка для аноніма ў `TrackLikesProvider`.

**Было ў тікеце:** user ↔ track; анонім без захавання — праз прапанову рэгістрацыі.

**Змест:** падзеі праслухоўвання (≥30 с, поўныя); анонімы — агрэгат па днях.

**AC:** парогі ў кодзе; без дубляў пры seek; мінімум ПД для анонімаў.

---

### SPEU-003 — Лічбы на песнях за 7 дзён

**Змест:** лайкі і праслухоўванні за 7 дзён на картках/старонках.

**Заўвага:** лайкі ў БД ўжо ёсць — засталося падлічваць і паказваць агрэгаты.
