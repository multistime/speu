# Дакументацыя праекту Speu (PM)

Гэта каталог для **агульнага апісання**, **роадмэпа** і **тікетаў**. Рэдагуйце `.md` файлы тут — змены адлюстроўваюцца на старонцы адмінкі **Праект** (`/admin/project`): усе раздзелы на адным скроле, з зместам і спасылкамі на блокі.

| Файл | Прызначэнне |
|------|-------------|
| `OVERVIEW.md` | База ведаў: мэта, стэйкхолдэры, межы прадукту, тэрміны |
| `ROADMAP.md` | Планы па часе (квартал / месяц / тыдзень) |
| `TICKETS.md` | Бэклог і актыўныя задачы (ID, статус, апісанне) |

Парада: для новых задач капіруйце шаблон з `TICKETS.md`.
